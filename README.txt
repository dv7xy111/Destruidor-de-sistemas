🧨 destruidor.sh - EXEMPLO TEÓRICO

Este projeto contém um exemplo de script com o comando 'rm -rf /',
usado unicamente para estudo e demonstração do que NÃO FAZER em Shell Script.

⚠️ NÃO EXECUTE o arquivo destruidor.sh
Ele contém um comando que, se rodado, apaga completamente o sistema Linux.

Incluído neste pacote:
- destruidor.sh -> Script destrutivo (não execute!)
- LICENSE.txt   -> Termos de uso e alerta
- README.txt    -> Explicação geral
